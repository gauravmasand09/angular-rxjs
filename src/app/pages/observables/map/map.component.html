Map Operator

Map

Map operator using .map() inside pipe is very useful in cases when want to modify the response. Suppose, you have a list of integers but instead you want integres with multiplication by 10. you can do like this -> .map(x => x * 10), after this when subscribe is called the resulting output will be : 10,20,30 elContainer1
Note: map operator changes data before subscription whereas tap can't change the data before subscribing. (Learn more)



Ex: 01 Map
Code: map((resp) => "Video "+(resp+1))
